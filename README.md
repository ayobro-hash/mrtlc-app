## 🚀 Architectural Capabilities

Nexus 4.0 now natively interprets the full core dictionary of Roblox runtime engines, delivering full quad-format support:

*   **`.rbxmx` (XML Model)** — 100% Fully Supported with deep DOM property parsing.
*   **`.rbxm` (Binary Model)** — 100% Fully Supported with symmetrical binary structure mapping.
*   **`.rbxlx` (XML Place)** — **[NEW]** Fully Supported via an integrated Asset Filter Shield that bypasses empty backend services to isolate physical place objects.
*   **`.rbxl` (Binary Place)** — Read capability active (Note: Large production place files may challenge mobile browser memory allocations).

## ✨ Features

*   **Omni-Class Engine Registry:** Deep-scans files for over 100+ native Roblox engine classes, including spatial geometry (`Part`, `MeshPart`), physics constraints (`Motor6D`, `Weld`), UI frameworks (`ScreenGui`, `Frame`), visual environments (`ParticleEmitter`, `Highlight`), and scripts.
*   **Dual-Pane Property Inspector:** Real-time extraction and split-view display of key asset data profiles (`ClassName`, `Size`, `AssetID`, `Source`).
*   **Luau Structure Builder Pipeline:** Automatically reconstructs raw asset metadata into highly accurate, standalone generation scripts ready for execution.
*   **Mobile-Optimized Engine Shield:** Built-in safeguards automatically throttle and paginate large lists over 400 assets to prevent browser crashes on self-hosted local mobile nodes.
