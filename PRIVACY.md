# Privacy Policy for MRTLC // CORE

Last updated: June 13, 2026

## Overview

MRTLC // CORE is designed to process Roblox files locally on the user's device. We prioritize transparency and privacy.

## Data Collection

MRTLC does not collect, store, sell, or share personal information.

## File Processing

Files selected by the user are processed locally on the device.

MRTLC does not upload files to external servers or cloud services.

## Internet Usage

MRTLC does not require an internet connection for its core functionality.

## Analytics and Tracking

MRTLC does not use analytics, advertising SDKs, telemetry, or tracking technologies.

## Third-Party Services

MRTLC does not integrate third-party services that collect user data.

## Open Source

The source code is publicly available for inspection:

https://github.com/ayobro-hash/mrtlc-app

## Changes

This Privacy Policy may be updated as the project evolves.

## Contact

Questions or concerns may be submitted through the project's GitHub repository.
